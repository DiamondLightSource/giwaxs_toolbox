GIWAXS toolbox

This package is a collection of python based tools to help with visualising and analysing Grazing Incidence Wide Angle X-ray Scattering (GIWAXS) data collected at Diamond Light Source on beamline I07.

The initial processing and mapping step is handled by the fast_rsm package, and this toolbox is designed to work with the output from that package. The toolbox includes a collection of interactive widgets to help explore and visualise GIWAXS datasets, as well as tools for extracting datasets for further analysis including fitting peaks in 1D datasets.
