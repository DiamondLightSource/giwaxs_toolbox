GIWAXS toolbox
===================

This package is a collection of python based tools to help with visualising and analysing Grazing Incidence Wide Angle X-ray Scattering (GIWAXS) data collected at Diamond Light Source on beamline I07



Accessing the giwaxs_toolbox
-------------------------------

   #. `diamond_guide`_ : Guide on accessing the giwaxs_toolbox on the Diamond filesystem 
   #. `local_install`_ : If you are familiar with setting up and using python environments you use the following instructions for installing a local version of giwaxs_toolbox



Guides on using the giwaxs_toolbox
-----------------------------------------

   `walkthrough guides`_ : Walkthroughs with examples on using the various options available within the giwaxs_toolbox


Training videos
-------------------------

   `training videos`_ : A collection of training videos on how to use the various features of the giwaxs_toolbox, and how to create datasets to use with the toolbox.
